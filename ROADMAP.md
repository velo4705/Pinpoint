# 🛰️ Pinpoint Milestone Roadmap

This roadmap outlines the journey from a prototype to a "perfect" production-ready AI geolocation engine.

## 🟩 Phase 1: Foundation (Completed - v1.0.0)
- [x] Initial Project Structure (Frontend/Backend)
- [x] Basic UI with MapLibre integration
- [x] Neural Engine foundation (CLIP Model)
- [x] Core "Scan" API
- [x] Real-world vs. Gaming mode architecture
- [x] Multi-Model Selector (Automatic, Gemini, Groq, GitHub, Local AI Engine)
- [x] Local Storage Operator History tray (quick recall)
- [x] Minecraft Seed-DNA Triangulation & block coordinates geoguessing
- [x] Custom Spectrum Accent changer (Red, Blue, Green, Purple, Orange)
- [x] Left-Wing Sidebar layout with central fluid-morphing telemetry console
- [x] Visual scan thumbnails & glowing neon Neural Confidence metrics

## 🟨 Phase 2: Neural & Privacy Refinement (Short Term)
- [x] **Advanced Geolocation**: Move beyond landmarks to "GeoGuessr" style environmental detection (flora, weather, soil).
- [ ] **EXIF Data Scrubber**: Automatically strip privacy-invasive metadata before processing.
- [ ] **Synthetic Scenery Shield**: Detect and label AI-generated imagery/deepfakes utilizing C2PA content credential checks.

## 🟦 Phase 3: Gaming & Open World (Mid Term)
- [ ] **Vector Database Integration**: Fully implement ChromaDB for game map matching.
- [ ] **Map Tiler**: A tool to convert high-res game map images into interactive map tiles.
- [ ] **Game Preset Library**: Built-in support for GTA V, RDR2, Genshin Impact, and Elden Ring.
- [ ] **Player Location HUD**: Directly query coordinates (where permitted) from active games.

## 🟪 Phase 4: Intelligence & Community (Long Term)
- [ ] **Collaborative Tactical Boards**: Multi-operator map sharing allowing remote teams to share pins, draw vector lines, and coordinate geoguessing sessions in real-time.
- [ ] **Crowdsourced Intelligence**: Allow users to contribute verified "Landmark" data to the open-source database.
- [ ] **Privacy Vault**: Optional end-to-end encryption for stored history.
- [ ] **Mobile App (PWA)**: Optimized "Scan-on-the-go" mobile interface.

## 🚀 Phase 5: The "Perfect Product" (Polishing)
- [ ] **Real-time Scan**: "Live" camera feed geolocation.
- [ ] **Offline Mode**: Downloadable local models for disconnected usage.
- [ ] **Developer API**: Allow other games/apps to query Pinpoint's geolocation engine.
- [ ] **Interactive 3D Flight Overlays**: Fly-over visualization grids mapping terrain elevations once a coordinate is locked.
